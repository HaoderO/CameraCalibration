Hidden Shortcuts

You may see most of the program's shortcuts by playing with the menus (if there is a shortcut for given menu item, it is shown at the right part of the menu). However, there are some key combinations that are not visible anywhere. They are listed in the table below.

Please, remember: All shortcuts that include the "Command" button on Mac are using the "Alt" button on Windows and vice versa. I made this that way to keep things as close as possible for my fingers between the different operating systems.

Move / Resize Ruler
Arrow Keys Move ruler 1 pixel left/up/right/down (can be modified from Preferences / Shortcuts)
Cmd / Alt + Arrows Fast Move (100px by default; can be modified from Preferences / Shortcuts)
Shift + Arrows Resize ruler (1pixel by default; editable in Preferences dialog)
Cmd/Alt + Shift + Arrow Resize faster (10px step by default)
Cmd/Alt + Mouse Wheel Resize Ruler
Shift + Mouse Wheel If the mouse is over figure, will resize figure;
If mouse cursor is over the ruler (not on flying figure) -- will resize ruler
Flying Figure
I, J, K, L Move Flying Figure 1px up / left / down / right (for fine-positioning)
A, Z, Rotate Flying Figure
S, X Rotate faster
D, C Rotate -- 45 degree
F, V Rotate -- 90 degree
Cmd/Alt + Shift + Mouse Wheel Rotate Figure
Cmd/Alt + Mouse If you press Alt (Cmd) while dragging control point, you'll prevent figure rotation (otherwise you resize and rotate in the same time); If you check the "Revert Alt Behavior" from the "Flying Figures" menu, you'll get the opposite behavior (by default rotation will be locked and Alt will unlock the rotate on resize feature)
Shift + Mouse To rotate figure on 45 degree intervals, pres and hold Shift key while dragging control point
Other
Double click Full Screen mode On / Off
P Start/Stop live focus (if live focus is On and you have opened more than one rulers, the ruler under the mouse will get the focus without clicking the left button)
