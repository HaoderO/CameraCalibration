Settings / Activation

If you want to continue using this tool after the trial period expiration, you must activate it.

To activate the program:

  • go to "Preferences" and click the "Program" tab
  • Paste your activation code in the respective field (the server sends you the activation code right after you buy the program)
  • Click "Activate" and wait few seconds until the program contacts the activation server

If you get an error message, please check your internet connection. If you are behind a firewall (or if you are using program like Zonealarm or LittleSnitch) -- be sure to grant internet access to the program.

If the program says that your activation code is no longer valid -- check your email. That can happen if for some reason the original activation code has been disabled. In such case new activation code will be send to the email address used at the time of purchase. If you didn't get this new activation code -- please, contact me as soon as possible (remember to mention the email/name used at the time of the purchase).

The activation requires internet access. Once activated, program can work offline (but you will need to re-activate if you reinstall the operating system)

Before selling or reinstalling your computer, or if you want to delete the rool, feel free to deactivate the program (there is a "Deactivation" button available on the same activation page).

On the same page you will see one "Auto check for updates" check box. By default Screen Ruler will silently check for new versions from time to time and to show you one-time notification message when new version becomes available. Uncheck this to disable automatic check for updates.

Finally, while we are still on the activation page, few words on software piracy:

Today everyone and his mother can find an illegal copy of any software. That is not big problem for me (the only thing I lose is potential sale) but could be really serious problem for this tool itself. If there are no sales, I'll lose my interest. If I lose my interest, the future development will fade out and the tool will eventually pass away. So, please, do not distribute your activation code to people who don't deserve it.
