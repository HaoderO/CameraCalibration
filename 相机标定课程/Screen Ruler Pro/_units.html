Screen Ruler Help

Settings / Units

You can edit / add Units to the ruler. To do that, just open the "Preferences" window and go to "Units".

For each unit, the following data must be entered:

  • Unit -- this is the name of the unit (the text you will see when changing units from the menu)
  • Abbrev. -- this must be 2 characters long (will be shown on ruler readings)
  • Units per Inch -- All unit conversions are calculated on units per inch basis.( For example, 1 in = 2.54 mm, so the Units per Inch value for Centimeters unit must be 2.54)
  • Ticks per Unit -- this value tells the ruler how many ticks to draw on the rulers per one unit. (For example: if you set 10 ticks per unit, there will be 10 ticks (small lines) drawn for every unit shown on the rulers)
    Note that the number of ticks per unit can lead to ugly result on screen. If you have too many ticks per unit while the unit sonsists of small number of pixels, you will not get ticks but one single wide line. Hint: take a look on the "Points" unit. To avoid drawing too many thick lines I have set this to 0.1

Step by step example: Adding new Unit

Let's add Picas to the list of units. A "pica" is a typographicmeasurement unit corresponding to 1/6 of an inch. So we set these values:

Unit: Abbrev.: Units per Inch: Ticks per Unit:
Picas pc 6 1

Now set Units to "Picas" and see the result.
