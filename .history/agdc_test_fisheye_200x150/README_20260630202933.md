## Mesh 表规格

### 数据格式 (36-bit per node)

```
┌──────────────────┬──────────────────┐
│   v[17:0]        │   u[17:0]        │
│   bits [35:18]   │   bits [17:0]    │
└──────────────────┴──────────────────┘
```

- **u12.5 signed**: 1 LSB = 1/32 px
- **SRAM 寻址**: `addr = iy × (mesh_cols+1) + ix`
- **加载文件**: `agdc_mesh_16x12_36bit_in_u64.bin`（**不是** `uv_fixed.bin`）

### 网格坐标范围

| 网格 | tile | u 范围 | v 范围 | 节点 |
|------|------|--------|--------|------|
| 16×12 | 8×8 | [67.7, 131.9] | [37.8, 112.0] | 17×13 = 221 |
| 32×24 | 4×4 | [67.7, 131.9] | [37.8, 112.0] | 33×25 = 825 |

> 所有节点均在鱼眼图像 [0,200]×[0,150] 内，有效比例 100%。

## 硬件约束分析

```
单行 v_in 极差 max = 74 px ≈ line_buf_depth (典型值 70)
逐行 vspan (相邻行位移) ≈ 0.03 px  ← 不是瓶颈
```

**结论: 三个分辨率中最接近在线模式可行的配置。** 单行 v_in 极差仅 74 px。

### Slice + ROI 降低行缓冲需求

| ROI 宽度 | 单行 v_in 极差 | line_buf≤70 时最大 slice_h | passes |
|---------|-------------|--------------------------|--------|
| 128 (全宽) | 74 px | 临界 | — |
| 112 | 65 px | **全高 96 行** | **1** |
| 96 | 56 px | 全高 96 行 | 1 |

> **ROI_w ≤ 112 即可在线模式单 pass 出图。**

## 推荐寄存器配置

### 离线模式 (16×12 tiles, NV12 格式)

```
// ============================================================
// Step 1: 初始化
// ============================================================
WR AGDC_BASE + 0x04 = 0x00000000   // CTRL: enable_rectify=0

// ============================================================
// Step 2: 配置寄存器组
// ============================================================

// --- 帧尺寸 ---
WR AGDC_BASE + 0x08 = 0x009600C8   // INPUT_SIZE:  h=150 (0x96), w=200 (0xC8)
WR AGDC_BASE + 0x0C = 0x00000000   // ROI_ORIGIN:  roi_x=0, roi_y=0
WR AGDC_BASE + 0x10 = 0x00600080   // OUTPUT_SIZE: h=96 (0x60), w=128 (0x80)

// --- 网格 ---
WR AGDC_BASE + 0x14 = 0x00000C10   // MESH_GRID: mesh_rows=12, mesh_cols=16

// --- 格式 ---
WR AGDC_BASE + 0x18 = 0x00000000   // FORMAT_CTRL: NV12 in/out

// --- 总线地址 (16B 对齐) ---
WR AGDC_BASE + 0x1C = <src_y_addr>     // SRC_Y_BASE_ADDR
WR AGDC_BASE + 0x20 = <src_y_addr + 0x00007530>   // SRC_UV_BASE_ADDR (Y + 30000)
WR AGDC_BASE + 0x24 = <dst_y_addr>     // DST_Y_BASE_ADDR
WR AGDC_BASE + 0x28 = <dst_y_addr + 0x00003000>   // DST_UV_BASE_ADDR (Y + 12288)

// --- 行跨度 ---
WR AGDC_BASE + 0x2C = 0x000000C8   // SRC_STRIDE: 200 bytes/row
WR AGDC_BASE + 0x30 = 0x00000080   // DST_STRIDE: 128 bytes/row

// --- 中断 ---
WR AGDC_BASE + 0x34 = 0x00004646   // ISP_LINE_BUF_CFG: 离线模式默认
WR AGDC_BASE + 0x40 = 0x00000001   // INT_MSK: frame_done

// ============================================================
// Step 3: 启动
// ============================================================
WR AGDC_BASE + 0x04 = 0x00000005   // CTRL: enable_rectify=1 + start_frame=1

// Step 4: 等待 busy=0 或 frame_done 中断
```

### 在线模式 (ROI_w=112)

若硬件支持在线模式，缩小 ROI 宽度即可单 pass：

```
WR AGDC_BASE + 0x08 = 0x00960070   // INPUT_SIZE ROI: h=150, w=112
WR AGDC_BASE + 0x10 = 0x00600070   // OUTPUT_SIZE:    h=96,  w=112
WR AGDC_BASE + 0x2C = 0x00000070   // SRC_STRIDE: 112 bytes/row
WR AGDC_BASE + 0x30 = 0x00000070   // DST_STRIDE: 112 bytes/row
```
