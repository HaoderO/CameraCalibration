## Mesh 表规格

### 数据格式 (36-bit per node)

```
┌──────────────────┬──────────────────┐
│   v[17:0]        │   u[17:0]        │
│   bits [35:18]   │   bits [17:0]    │
└──────────────────┴──────────────────┘
```

- **u12.5 signed**: 1 LSB = 1/32 px
- **SRAM 寻址**: `addr = iy × (mesh_cols+1) + ix`
- **加载文件**: `agdc_mesh_16x12_36bit_in_u64.bin`（**不是** `uv_fixed.bin`）

| 网格   | tile    | u 范围        | v 范围        | 节点          |

## 硬件约束分析

```
单行 v_in 极差 max = 539 px >> line_buf_depth (典型值 70)
逐行 vspan (相邻行位移) ≈ 0.24 px  ← 不是瓶颈
```

**结论: 离线模式必须。** 高分辨率 (1448×1086) 下 v_in 极差达到 539 px，
是所有配置中行缓冲需求最大的。

### Slice + ROI 降低行缓冲需求

| ROI 宽度     | 单行 v_in 极差 | line_buf≤70 时最大 slice_h | passes |
| ------------ | -------------- | -------------------------- | ------ |
| 912 (全宽)   | 539 px         | 不可行                     | —      |
| 456          | 270 px         | 不可行                     | —      |
| 228          | 135 px         | 不可行                     | —      |
| 160          | 95 px          | 32 行                      | 22     |
| 120          | 71 px          | 40 行                      | 18     |

## 推荐寄存器配置

**离线模式, 16×12 tiles, NV12 格式:**

```
// ============================================================
// Step 1: 初始化
// ============================================================
WR AGDC_BASE + 0x04 = 0x00000000   // CTRL: enable_rectify=0

// ============================================================
// Step 2: 配置寄存器组
// ============================================================

// --- 帧尺寸 ---
WR AGDC_BASE + 0x08 = 0x043E05A8   // INPUT_SIZE:  h=1086 (0x43E), w=1448 (0x5A8)
WR AGDC_BASE + 0x0C = 0x00000000   // ROI_ORIGIN:  roi_x=0, roi_y=0
WR AGDC_BASE + 0x10 = 0x02AC0390   // OUTPUT_SIZE: h=684 (0x2AC), w=912 (0x390)

// --- 网格 ---
WR AGDC_BASE + 0x14 = 0x00000C10   // MESH_GRID: mesh_rows=12, mesh_cols=16

// --- 格式 ---
WR AGDC_BASE + 0x18 = 0x00000000   // FORMAT_CTRL: NV12 in/out

// --- 总线地址 (16B 对齐) ---
WR AGDC_BASE + 0x1C = <src_y_addr>     // SRC_Y_BASE_ADDR
WR AGDC_BASE + 0x20 = <src_y_addr + 0x0017FEB0>  // SRC_UV_BASE_ADDR (Y + 1572528)
WR AGDC_BASE + 0x24 = <dst_y_addr>     // DST_Y_BASE_ADDR
WR AGDC_BASE + 0x28 = <dst_y_addr + 0x000984C0>  // DST_UV_BASE_ADDR (Y + 623808)

// --- 行跨度 ---
WR AGDC_BASE + 0x2C = 0x000005A8   // SRC_STRIDE: 1448 bytes/row
WR AGDC_BASE + 0x30 = 0x00000390   // DST_STRIDE: 912 bytes/row

// --- 中断 ---
WR AGDC_BASE + 0x34 = 0x00004646   // ISP_LINE_BUF_CFG: 离线模式默认
WR AGDC_BASE + 0x40 = 0x00000001   // INT_MSK: frame_done

// ============================================================
// Step 3: 启动
// ============================================================
WR AGDC_BASE + 0x04 = 0x00000005   // CTRL: enable_rectify=1 + start_frame=1

// Step 4: 等待 busy=0 或 frame_done 中断
```
